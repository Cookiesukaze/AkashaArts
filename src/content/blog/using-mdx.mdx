冒泡排序是一种简单且直观的排序算法。它通过重复遍历要排序的列表，依次比较相邻的元素并交换顺序不对的元素来达到排序的目的。

## 为什么选择冒泡排序？

冒泡排序的基本思想是将最大的元素逐步“冒泡”到列表的末尾。虽然冒泡排序的时间复杂度较高（最坏和平均情况下是 O(n²)），但由于其实现简单，适合初学者学习和理解排序算法的基本概念。

## 示例

下面是一个简单的冒泡排序的实现示例：

```js
function bubbleSort(arr) {
  let n = arr.length;
  for (let i = 0; i < n - 1; i++) {
    for (let j = 0; j < n - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        // 交换相邻元素
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
      }
    }
  }
  return arr;
}

// 测试冒泡排序函数
const array = [64, 34, 25, 12, 22, 11, 90];
console.log('排序前:', array);
console.log('排序后:', bubbleSort(array));
